# PYTORCH ESSENTIALS

This repo covers the essentials functionalities/concepts of the Pytorch library. This is to be used for learning purposes.

## Setup

Create a venv environment and then activate the environment

```bash
pytorch-essentials> python -m venv .venv
pytorch-essentials> source .venv/bin/activate
```

Install the requirements

```bash
(.venv)pytorch-essentials> pip install -r requirements.txt
```
